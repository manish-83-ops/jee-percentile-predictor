# 📊 JEE Percentile Predictor

A Machine Learning web app that predicts JEE percentile based on marks and difficulty level.

## 🚀 Features
- Predict percentile using ML model
- Simple Streamlit UI
- Random Forest Regressor backend

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
